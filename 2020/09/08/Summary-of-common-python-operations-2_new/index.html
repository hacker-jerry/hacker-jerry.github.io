图片大小与像素

有时候光设置 figsize 在绘制多子图时，整个画布的大小还是没有改变，所以需要更改 figure 的 dpi 让它大一些（只是看起来方便）

图片风格覆盖

由于 seaborn 的库实现是基于 matplotlib 的，所以可以直接对 sns 进行设置风格，从而覆盖原 plot 属性。

  • darkgrid 暗色网格（默认）
  • whitegrid 白色网格
  • dark 暗色背景
  • white 白色背景
  • ticks 应该是四周都有刻度线的白背景

更加精细化定制

如果您想要定制 seanborn 的样式，可以将参数字典传递给 axes_style()和 set_style()的 rc 参数。注意，只能通过该方法覆盖样式定义的一部分参数。(然而，更高层次的 set()函数接受任何 matplotlib 参数的字典)。
