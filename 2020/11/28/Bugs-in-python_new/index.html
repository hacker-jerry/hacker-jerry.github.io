跨目录调用模块

对于这样一个目录结构的项目，我们需要知道：
Python 解析器对模块位置的搜索顺序：

  1. 当前目录。
  2. PYTHONPATH 的每个目录。
  3. 查看 Python 安装时的默认包路径。例如：/usr/lib64/python2.6/site-packages。

    可以通过查询sys.path查看模块搜索路径。

在子包（package)中引入根目录

  • 将项目的根目录加到 PYTHONPATH 变量中

  • sys.path.append(“..”)
